# 🎬 Movie Booking API

A FastAPI-based backend for managing movies, theaters, shows, and bookings.

## 🚀 Features
- Add and fetch **Movies**
- Manage **Theaters**
- Schedule **Shows**
- Create and view **Bookings**
- MySQL database integration with SQLAlchemy

## 🛠️ Tech Stack
- **FastAPI** (Backend framework)
- **SQLAlchemy** (ORM)
- **MySQL** (Database)
- **Uvicorn** (ASGI server)
